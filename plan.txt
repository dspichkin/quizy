Сервис онлайн уроков для тестирования знаний студентов

основные возможности: создания и публикация уроков ввиде последовательности вопрос/ответов


1 Заглавная страница
2 Страница уроков (разные интерфейсы для преподователей и для студентов)
	обе страницы должны состоять из списков но нужно предусмотреть начальное состояние
	страницы когда списки пустые
	и состояние когда список содержить больше кол-во элементов

	а. Для преподователей основным элементов является список своих уроков и действий на ними
		основные действия
		- создать новый урок
		- редактировать
		- искать
		- удалить
		- назначить на студента
		- опубликовать

		так же нужно сразу отображать
		на кого назначен урок
		статистику прохождения уроков
			- дата прохождения
			- пройден или нет
			- кол-во попыток

	б. Для студенов основным элементом список назначенных на него уроков
		Действия
		- начать урок
		- отвергнуть
		- переместить в архив

	нужно учесть что один человек может быть как студентом так и перподователем
	то есть нужно как то удобно переключаться между эти интерфесами (как вариант меню с табами)

3 Страница создания/редактрования урока
	редактируемые элементы
		- название урока
		- описание урока
		- разрешить/запретить урок
		- создание новой страницы
		- удаление новой страницы
		- номер страницы в уроке
		- текст страницы/вопроса
		- варианты ответа для вопроса

4 Страница прохождения урока
	Вывод текущей страницы
	Навигация вперед (назад - как опция)
